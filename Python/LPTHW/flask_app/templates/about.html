This is a showcase Student classroom project that connects data management using SQL (SQLite / SQLAlchemy), Python + Flask for web application management and basic html websites. The project involves functionalities of:

  • Creating a database with various data types
  • Displaying all data positions in a website
  • Displaying single data position in a website
  • Adding new data positions on a website
  • Editing data positions on a website
